# For AI - Input Folder

Drop files here for the agent to process:

- **Screenshots** of products from websites
- **PDFs** of product specifications or comparisons
- **Text files** with links or notes
- **Images** of products you're considering

The agent will read these when evaluating your purchase options.

## Organization

You can organize by purchase:
```
for-ai/
├── 5g-router/
│   ├── option1-screenshot.png
│   ├── option2-screenshot.png
│   └── spec-comparison.pdf
└── office-chair/
    └── ergonomic-options.png
```

Or just drop files in the root for the current purchase.
