# From AI - Output Folder

The agent places generated outputs here:

- **Research summaries** - Product evaluations and comparisons
- **Recommendations** - Final purchase recommendations
- **Reports** - Detailed analysis documents

## Typical Outputs

After a purchase decision workflow:
```
from-ai/
└── 5g-router-2024-01/
    ├── research.md        # All products evaluated
    ├── recommendation.md  # Final recommendation
    └── comparison.md      # Side-by-side if requested
```
